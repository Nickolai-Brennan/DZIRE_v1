# Skills Docs — DZIRE_v1

## Overview
Skills are reusable, trigger-focused capability packages. Each skill lives in `skills/[skill-name]/SKILL.md`.

## Required Domain Skills

| Skill | Folder | Trigger |
|---|---|---|
| `project-planner` | `skills/project-planner/` | Plan a new project |
| `stack-verifier` | `skills/stack-verifier/` | Verify environment and tools |
| `frontend-builder` | `skills/frontend-builder/` | Add pages, components, routes |
| `backend-builder` | `skills/backend-builder/` | Add routes, services, models |
| `database-designer` | `skills/database-designer/` | Design tables, migrations, seeds |
| `api-designer` | `skills/api-designer/` | Define REST/GraphQL contracts |
| `documentation-generator` | `skills/documentation-generator/` | Write or update docs |
| `eval-runner` | `skills/eval-runner/` | Create and run agent evals |
| `deployment-planner` | `skills/deployment-planner/` | Deploy to production |
