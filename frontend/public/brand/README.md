This directory holds all official brand asset files for the DZIRE platform.

## Expected Files

| File | Description |
|------|-------------|
| `logo-primary.svg` | Full primary logo (wordmark + icon) |
| `logo-horizontal.svg` | Horizontal lockup |
| `logo-icon.svg` | Stand-alone icon mark |
| `logo-light.svg` | Light-mode version |
| `logo-dark.svg` | Dark-mode version (primary usage) |

## Logo Usage Rules

1. Maintain clear space equal to the height of the logo icon on all sides.
2. Do **not** stretch, rotate, or distort the logo.
3. Do **not** place the primary logo on backgrounds with less than 4.5:1 contrast ratio.
4. Use `logo-icon.svg` for small placements (favicon, avatar, app icon).
5. Use `logo-dark.svg` on dark backgrounds (default for DZIRE UI).
6. Use `logo-light.svg` only on white/light promotional materials.
7. Use `watermark.svg` only on branded photography and video content.
