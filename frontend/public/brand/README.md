# DZIRE Brand Assets

This folder contains all official DZIRE brand assets.

## Required Files (add when available)

| File                  | Description                        |
|-----------------------|------------------------------------|
| `logo-primary.svg`    | Primary full logo (dark bg)        |
| `logo-horizontal.svg` | Horizontal lockup                  |
| `logo-icon.svg`       | Icon mark / symbol only            |
| `logo-light.svg`      | Light version for dark backgrounds |
| `logo-dark.svg`       | Dark version for light backgrounds |
| `favicon.svg`         | Favicon (32×32 or scalable)        |
| `watermark.svg`       | Transparent watermark for media    |

## Usage Rules

- Maintain clear space equal to the height of the logo icon on all sides.
- Do not stretch, rotate, or apply effects to the logo.
- Do not use on low-contrast backgrounds.
- Use the icon mark for small spaces (< 32px height).
- Use the full logo for headers and official materials.
- The watermark is only for branded media assets (thumbnails, graphics).
